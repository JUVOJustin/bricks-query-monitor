== Description ==

Bricks Query Monitor seamlessly integrates with Bricks Builder to add loop monitoring capabilities to the Query Monitor plugin. This tool is essential for developers looking to optimize and debug their Bricks Builder loops effectively.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The plugin works immediately after activation; no further setup required.

== Frequently Asked Questions ==

= What is the purpose of this plugin? =

This plugin allows developers to monitor and debug Bricks Builder loops within the WordPress Query Monitor plugin.

= Does this plugin require Bricks Builder and Query Monitor? =

Yes, it requires both Bricks Builder and Query Monitor to function correctly.
